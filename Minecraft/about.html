About This Project

This website is a Minecraft-inspired crafting simulator developed as a front-end web development project using HTML, CSS, and JavaScript. It recreates the game's 3×3 crafting system, allowing users to drag and drop items into a crafting grid, detect valid recipes, and display the crafted result through an interactive and user-friendly interface.


The project was designed to strengthen my understanding of web development concepts such as DOM manipulation, drag-and-drop functionality, event handling, and responsive user interface design. Artificial intelligence (AI) was also used as a learning and development tool to assist with planning, coding, debugging, and improving the overall design while ensuring I understood how each feature was implemented.
